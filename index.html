Side Projects

Machine Learning Baseball (2018)

View Source Code Visit Demo Site

Machine Learning Baseball is a Python web application that employs machine learning techniques to predict pitcher vs. batter matchups of Major League Baseball games. This application was my final project for Harvard CS50, which I completed in Spring 2018 through the Harvard Extension School. Beginning with a sparse matrix of available pitcher vs. batter statistics, Low Rank Matrix Factorization was used to generate a fully populated matrix which predicts statistics for every possible matchup. A Python Flask web application was created to display the resulting data in an easy-to-use interface.

First to Know (2016 / 2017)

View Source Code Visit Demo Site

First to Know is a Ruby on Rails web application where users can subscribe to receive text message alerts when a selected twitter user tweets a certain phrase. For example, perhaps you want to be the first to know when Papa John's is giving away free pizza. You can create a subscription to Papa John's twitter account for the phrase 'free pizza'. As soon as Papa John's tweets anything containing the string 'free pizza', a text message will be sent to the user's provided phone number within minutes.

Swell Journal (2015 / 2016)

Swell Journal is a Ruby on Rails web application that allows users to document and share their surf travels with friends. The application uses geolocation services, Google Maps, and an oceanographic weather API to create a snap-shot of the conditions at the time of the journal entry.

Chchchanges (2016)

View Source Code

Chchchanges is a Ruby gem that makes it easy for a team of developers to keep a consistent, detailed Changelog. With chchchanges, users can create CHANGELOG entries from the command line. Each entry is saved as an individual .json file which prevents CHANGELOG merge conflicts.
