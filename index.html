Machine Learning Baseball (2018)

View Source Code

Machine Learning Baseball is a Python web application that employs machine learning techniques to predict pitcher vs. batter matchups of Major League Baseball games. Beginning with a sparse matrix of available pitcher vs. batter statistics, Low Rank Matrix Factorization was used to generate a fully populated matrix which predicts statistics for every possible matchup. A Python Flask web application was created to display the resulting data in an easy-to-use interface.

First to Know (2016 / 2017)

View Source Code

First to Know is a Ruby on Rails web application where users can subscribe to receive text message alerts when a selected twitter user tweets a certain phrase. For example, perhaps you want to be the first to know when Domino's is giving away free pizza. You can create a subscription to Domino's twitter account for the phrase 'free pizza'. As soon as Domino's tweets anything containing the string 'free pizza', a text message will be sent to the user's provided phone number within minutes.
